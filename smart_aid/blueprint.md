# Smart Aid Application Blueprint

## Overview

A Flutter application that provides smart assistance for medical situations. It will include features like a first aid guide, emergency contact information, and a personal medical profile.

## Features

*   **Home Screen:** A dashboard with quick access to other features.
*   **First Aid Guide:** A searchable guide for various first aid scenarios.
*   **Emergency Screen:** Displays emergency contacts and allows for quick calls.
*   **Profile Screen:** A place for users to store their medical information.

## Design

*   **Theme:** A clean and modern theme with a focus on accessibility.
*   **Navigation:** A bottom navigation bar for easy access to the main screens.
